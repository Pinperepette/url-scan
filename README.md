# url-scan
un semplice script per scrivere in un file tutti gli url di un sito
OPTIONS:

   -h      Show this message
   -t      Target
   -s      Time Sleep #(il tempo del comando sleep, che serve a non farsi bannare da google)
   -p      Set Scan # 3 livelli di ricerca
           1 (default scan)
           2 (Ghost scan)
           3 (Uber scan)
